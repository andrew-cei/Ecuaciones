# Ecuación Diferencial de decaimiento radioactivo
En este repositorio encontrarás una comparación entre dos tipos de soluciones a la ecuación del decaimiento radioactivo, una solución analítica y una numérica.

## Requisitos Previos
Para poder ejecutar el programa debes instalar los siguientes paquetes de python
* **Python 3.x**
* **Numpy**
* **Matplotlib**
* **Jupyterlan**

Sigue los pasos siguientes para echarlo a andar.

1. Clona el repositorio en tu computadora

1. Activa el entorno virtual

```
source env/bin/activate
```

1. Instala las dependencias mediante

```
pip install -r requirements.txt
```

1. Ejecuta Jupyterlab mediante

```
jupyter lab
```

1. Abre la dirección local en tu nevegador

1. Sigue los pasos del cuaderno de python
